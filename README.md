# Boots and Muskets
 
STEP PER COMPILARE:

Requisiti:
    - GCC
    - CMake

LINUX (Testato e consigliato):
    1. Aprire directory nel terminal
    2. ```sh buildAndRun.sh```

Windows:
    1. Aprire cartella nel prompt dei comandi
    2. ```cmake -S . -B build```
    3. ```cmake --build```
    4. Aprire ```build/bin/debug/```
    5. Eseguire bootsAndMuskets.exe
